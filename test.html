Portfolio

Hydrosense provides a communication protocol and back-end server infrastructure to enable any Arduino or Raspberry Pi to easily collect and publish hydroclimatic data. This data can then be queried, compared, and downloaded by researchers and hobbyists alike. See what's new on our Github repo!

At Sikorsky Aircraft, a Lockheed Martin company, I was tasked with updating the database used in their estimating department. This involved decomposing a collection of Excel sheets into a relational model. I then built an Access based solution from this model. In order to future proof the system, I automated the system of polling an off-site database for up to date resources. To improve the system's ease of use, I added a frontend to the program. This GUI also provided functionality which could automate various tasks which are common in the estimating process.
